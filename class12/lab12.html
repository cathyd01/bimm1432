Q.7 What is the purpose of the arr.ind argument in the which() function call above? Why would we then take the first column of the output and need to call the unique() function? It returns the row and column indices with a value of zero, for us to identify which samples from which genes have zero expression. We need to call the unique function to get the unique gene names to remove both the control and experimental results of each gene.
